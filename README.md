# dotfiles

My personal configuration files.


## Installation

```bash
# Make sure GNU Stow (https://www.gnu.org/software/stow/) is installed:
sudo apt-get install stow

cd /path/to/dotfiles

./install.sh
```

```bash
git clone https://github.com/sumithemmadi/dotfiles.git
cd dotfiles
./install.sh
exit
```
